# Project Outputs

This folder contains sanitized project output files generated from the MATLAB EEG–ECG analysis pipeline.

## Contents

* `report/` - Final project report or summarized PDF documentation
* `tables/` - Sanitized Excel/CSV summary tables

## Privacy Note

The uploaded files do not contain raw EDF data, patient names, patient IDs, hospital IDs, or identifiable clinical metadata.

Subjects are represented only using anonymized labels such as `Subject_01`, `Subject_02`, `Subject_03`, and `Subject_04`.

Raw clinical recordings are not included in this repository.
